# Loops Campaign Performance

Each morning it reads opens, clicks, and replies — and posts which sequences are landing and which are quietly dying.

## Prerequisites
- A [Loops](https://loops.so) account with API access (grab a key from `app.loops.so/settings/api`)
- A Slack workspace where you can install the agent's bot and invite it to one or more channels
